Componentes básicos de un sistema de información.

Entrada

Conformada con la recopilación y captura de datos al sistema. Ejemplo:

  • El registro de las actas de calificaciones en la facultad.
  • Registro de un acta de escrutinio y cómputo en una elección.
  • El registro de los montos en una venta de medicamentos en una farmacia.


Procesamiento

Es realizar operaciones (manual o automatizada) sobre los datos de entrada para extraer la mayor cantidad de información útil para la organización. Las operaciones incluyen:

  • Almacenar para posterior uso.
  • Cálculos.
  • Comparar valores.
  • Realizar acciones alternas.


Salida

Conformada por toda la información útil lograda en el procesamiento. La Salida de un sistema puede ser la entrada de otro sistema. Ejemplos:

  • El historial académico de un alumno.
  • El reporte de compras realizadas un día antes. Este puede ser la entrada de un sistema de facturación.
  • Las gráficas de votos totales en un sistema de conteo.


Retroalimentación

Son los mensajes que emite el sistema para realizar cambios en la entrada o en la lógica de procesamiento. Un valor de entrada fuera de rangos permitidos, lanza un cuadro de diálogo. La retroalimentación de un sistema también surge cuando el estado de los datos que se encuentran en procesamiento dispara un evento en el que los administradores del sistema deben tomar una decisión. Ejemplo: El sistema detecta que un producto alcanzó el mínimo de existencias en inventario y lanza un mensaje (cuadro de dialogo o correo electrónico) para avisar el caso.
