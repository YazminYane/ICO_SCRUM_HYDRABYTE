MIS

Sistemas de Información Gerencial

  • Apoya a la gerencia con las ac/vidades de control, monitoreo y administra/va para la toma de decisiones.
  • Orientado a la Gerencia de nivel medio.
  • Sinte/zan e informan el comportamiento de el nivel opera/vo, tomando como base datos de ERP, o un TPS, o de otros sistemas de todas las áreas de la organización.
  • Permiten presentar un informe del estado de la organización en un intervalo de /empo predefinido.

La información de entrada de este sistema puede ser:

  • De sistemas TPS:
    • Sistemas de inventario.
    • Sistemas de planeación de producción.
    • Sistemas de Producción.
    • Sistema de contabilidad y finanzas.
    • Sistema de recursos Humanos.
    • Sistema de ventas.
    • Sistemas de Inteligencia de negocio.
  • De un Sistema de Planificación de Recursos (ERP), que son sistemas que integran a los anteriores.

Si lo vemos desde el punto de vista técnico:

  • Es una base de datos centralizada que almacena toda la información del estado de la compañía para que los administradores generen reportes y tomen decisiones.
  • No almacena toda la información del negocio, solo un resumen de cada área.
  • Procesa esta información para generar reportes.
